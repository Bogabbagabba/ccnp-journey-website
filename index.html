DATE: 2025-09-13 | CATEGORY: CCNP_ENTERPRISE | DIFFICULTY: ★★★☆☆
Today I tackled OSPF multi-area configurations and learned about LSA types. The concept of area border routers (ABRs) finally clicked when I visualized the topology using GNS3. Key insights:

• Area 0 (backbone) is mandatory for inter-area routing
• LSA Type 3 summary advertisements between areas
• Stub areas reduce routing table size significantly
IOS Configuration - OSPF Multi-Area
router ospf 100
 router-id 1.1.1.1
 area 10 stub
 network 10.1.1.0 0.0.0.255 area 0
 network 10.2.1.0 0.0.0.255 area 10
 area 10 range 10.2.0.0 255.255.0.0
!
interface fa0/0
 ip ospf priority 100
